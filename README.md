## ⚙️ Instalación

```bash
git clone https://github.com/aledash3/NumPy.git
cd NumPy
python -m venv venv
```

Activar entorno:

```bash
# Windows
venv\Scripts\activate

# Linux/Mac
source venv/bin/activate
```

Instalar:

```bash
pip install numpy
```
